MYSQL ERROR FIX you should use root no pass, 

SELECT user, host FROM mysql.user;

You will likely see something like:

root | %


CREATE the sail user for Docker (NO PASSWORD)
CREATE USER 'sail'@'%' IDENTIFIED BY '';

Grant permissions (IMPORTANT)
GRANT ALL PRIVILEGES ON *.* TO 'sail'@'%' WITH GRANT OPTION;
FLUSH PRIVILEGES;

Verify user now exists
SELECT user, host FROM mysql.user;

you will see
sail | %
